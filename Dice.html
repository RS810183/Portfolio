Dice

This is another project that I made in class. It shows a 12 dice in a 4 by 3 configuration. The hardest part of this project would probably have been making the dice class itself, and making sure that the pips will always be in the correct position relative to the dice. To fix the scaling with the dice I found the correct position for the pips for one dice, and then applied that to all the dice.
